🌿 EcoStep – Carbon Footprint Tracker

EcoStep is a modern, responsive web application that helps users estimate and visualize their annual carbon footprint based on their daily lifestyle. The project provides interactive charts, personalized insights, goal tracking, and AI-powered sustainability recommendations.

✨ Features

- 📊 Carbon footprint calculator
- 🚗 Track emissions across multiple categories:
  - Transport
  - Electricity
  - Food
  - Waste
  - Shopping
- 🎯 Set and monitor annual carbon reduction goals
- 📈 Interactive dashboard with:
  - Circular gauge
  - Progress bars
  - Pie chart
  - Trend visualization
- 🏆 Achievement badges and progress tracking
- 💡 Personalized carbon reduction tips
- 🤖 AI-powered sustainability insights (Claude AI)
- 📱 Mobile-first responsive design
- ⚡ Fast, lightweight, and works directly in the browser

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- SVG Charts
- Google Fonts
- Claude AI (for personalized insights)

📂 Project Structure

EcoStep/
│── index.html
│── README.md

🚀 Getting Started

1. Clone the repository

git clone https://github.com/your-username/ecostep.git

2. Open the project folder.

3. Open "index.html" in your browser.

No installation or build tools are required.

🎯 Purpose

The goal of EcoStep is to increase awareness about personal carbon emissions and encourage users to adopt more sustainable habits through data visualization, goal tracking, and AI-generated recommendations.
